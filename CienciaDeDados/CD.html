Tendências Centrais

As tendências centrais são utilizadas para representar um conjunto de dados com um único valor, sendo importante no estudo da estatística.

Moda

Em um conjunto de dados, é o elemento que mais se repete, pode ser:
No python, pode ser calculada utilizando o método mode().

  • Modal: Possui uma única moda
  • Bimodal: Quando um conjunto possui duas modas
  • Polimodal: Quando possui três ou mais modas
  • Amodal: Todos os elementos com a mesma frequência

Mediana

É o elemento central do conjunto quando colocado em ordem crescente.
No Python, pode ser calculada através do método median().

Média Aritmética

É calculada pela soma de elementos do conjunto dividido pela quantidade de elementos


No Python, pode ser calculada através do método mean().

Média Aritmética Ponderada

Na média aritmética ponderada, são atribuídos pesos para cada um dos valores. Quanto maior for o peso, maior será a influência daquele determinado dado no valor da média aritmética ponderada.

Percentil

É uma medida que divide uma amostra em ordem crescente em 100 partes, cada uma com uma porcentagem de dados aproximadamente igual. No Python, pode ser calculado através do método quantile()
