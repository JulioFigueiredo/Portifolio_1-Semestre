Estruturas de Decisão

If/Else:

If: Ela recebe um parâmetro entre (), se for verdadeiro, executa o bloco de instruções

                
                    int idade = 18;
                    if (idade>=18){
                        System.out.println("Maior de idade");
                    }

Else: Se a condição do If for falsa, é executado o bloco de instruções dentro do Else

                
                    int idade = 17;
                    if (idade>=18){
                        System.out.println("Maior de idade");
                    }
                    else {
                        System.out.println("Menor de idade");

                    }

Switch

A estrutura Switch permite que vários testes de condições sejam feitos, como se fossem vários ifs aninhados. Se nenhum dos cases for verdadeiro, o bloco Default é executado. 

                
                    int opcao = 3;
                    switch (opcao) {
                        case 1:
                            // instruções
                            break;
                        case 2:
                            // instruções
                            break;
                        case 3:
                            // instruções
                            break;
                        default:
                            // instruções
                            break;
                    }

Estruturas de Repetição

Estrutura While:

A estrutura While cria um loop enquanto uma condição é verdadeira: 

 
                
                    int i = 0;
                    while (i<10) {
                        System.out.println("Hello, world");
                        i++; // incrementa de 1 em 1
                        // esse código imprimirá hello world 10 vezes
                    }

Estrutura Do While:

A estrutura Do While faz com que o que está dentro de "do" seja executado pelo menos uma vez: 

                
                    int i = 0;
                    do {
                        System.out.println("Hello, world");
                        i++;
                        // será executado uma vez
                    }
                    while (i<=0);

Estrutura For:

A estrutura for funciona como o While (faz um loop), porém tem uma estrutura fixa como: 

                
                    for(int i=0; i<3;i++){
                        System.out.println("Hello, world");
                        // loop começa em 0
                        // incrementa de 1 em 1
                        // roda enquanto for menor que 3
                    }

Vetores e Matrizes

Vetores:

Vetores são variáveis que armazenas vários dados do mesmo tipo.
Para Acessá-los é necessário passar o índice dele 

                
                    // declaração do vetor
                    int [] vetor = {0, 10, 20, 30, 40, 50};
                    //vetores começam com índice 0
                    // printará o vetor da posição 2, ou seja, 20.
                    System.out.println(vetor[2]);

Para percorrer o vetor inteiro: 

                
                    // declaração do vetor
                    int [] vetor = {0, 10, 20, 30, 40, 50};

                    // percorrer através do for normal
                    for(int i = 0; i < vetor.length; i++){
                        System.out.println(vetor[i]);
                    }

                    // for each
                    for (int valor: vetor) {
                        System.out.println(valor);

                    }

A propriedade .length retorna o tamanho do Vetor (numero de posições).

Matrizes:

Matrizes são vetores com mais de uma dimensão,então elas tem linhas e colunas a serem percorridas 

                
                    // declaração de matrizes
                    int matriz [] [] = {{0, 100, 200}
                                        ,{300, 400, 500}};
                    // percorrer a matriz
                    // variavel i é linha
                    // variavel j é coluna
                    for (int i = 0; i < matriz.length; i++) {
                        for(int j = 0; j< matriz[0].length;j++) {
                            System.out.println(matriz[i][j]);
                        }
                    }

Funções

São blocos de instrução que, quando chamados, executam tudo que está nele, serve para evitar repetição de código por exemplo.
As funções podem ter parâmetros ou não, também podem ter retorno ou não (void) 

                
                    public class Main {
                        public static void main(String[] args) {
                            System.out.println(soma(5, 10));
                        }
                        public static int soma (int n1, int n2){
                            return n1+n2;
                        }
                    }

Funções Recursivas:

É um tipo de função que chama a si mesma

                
                    public class Main {
                        public static void main(String[] args) {
                    
                            int resultado = fatorial(5);
                            System.out.println(resultado);
                        }
                        public static int fatorial (int n1){
                            if (n1==1)
                                return 1;
                            else
                                return n1*fatorial(n1-1);
                        }
                    }
