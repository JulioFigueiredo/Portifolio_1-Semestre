Desenvolvimento Web

HTML

Nesta disciplina, começamos aprendendo os fundamentos da Linguagem de Marcação de Hipertexto HTML5 (HyperText Markup Language), sintaxe básica, tags e ver os resultados no navegador.

Algumas de suas principais tags:

  • head (normalmente são definidas as informações do documento html,
    importação de outros documentos, exemplo: tag title);
  • body (onde fica todo o corpo do documento, algumas tags: h1, h2, h6, p, a, img)

HTML Semântico

O HTML semântico tem o papel de trazer mais clareza aos documentos, descrevendo o conteúdo por meio de algumas tags como no exemplo.

CSS

Posteriormente, começamos a ver a linguagem de Estilização CSS3 (Cascading Style Sheets), para atribuição de estilos a página WEB escrita pelo no HTML5.
Essa linguagem permite definir estilos para elementos HTML, como cores, fontes, espaçamento entre elementos, layout de páginas, e muitos outros aspectos visuais.

JavaScript

Depois de aprender HTML5 e CSS3, começamos a aprender, enfim, uma Linguagem de Programação, o JavaScript, que é uma das linguagens mais utilizadas no mundo, ela é muito importante, pois sem ela, a página web fica estática. Ao utilizar JavaScript, é possível trazer dinâmica ao site como por exemplo trocar a cor de fundo da página quando um botão é pressionado. Apesar de ter aprendido a base de programação com Java, nos foi ensinado a sintaxe de JavaScript nesse período.

Aqui está um exemplo de um código que imprimirá "Hello, world" 5 vezes no console: 

                
                    let i = 0;

                    while (i<5){
                        console.log("Hello, world! ");
                        i++;
                    }

DOM

O DOM (Document Object Model) foi um tópico bem importante que aprendemos em JavaScript, com ele é possível pegar elementos do HTML por exemplo, e atribuir a eles uma variável no JS, assim é feito como citado acima, a mudança da cor de fundo do site quando um botão é presionado por exemplo.

Um exemplo: 

                
                    // pegando o documento html pelo ID dele
                    // atribui o documento html a uma variável
                    const botao = document.getElementById('btnTeste'); 

                    // o botão ao ser clicado, executa a função msg
                    botao.addEventListener('click', msg); 

                    // imprime um alerta na tela
                    function msg() {
                        alert('teste')
                    }
