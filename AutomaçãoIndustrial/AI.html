Durante o semestre, fizemos alguns e exercícios e pequenos projetos como um semáforo, onde pudemos aplicar nossos conhecimentos obtidos nesse período.

Exemplo:

Aqui está um exemplo de um código simples para ligar um led quando um botão for pressionado no arduíno: 

                    
                        bool indicador_verde = false; 
                        bool botao_pressionado = false;

                        void setup() {
                        pinMode(6, INPUT); // o botao está no pino 6 do arduíno
                        pinMode(4, OUTPUT); // o led está no pino 4 do arduíno
                        }

                        void loop() {
                        // a função digitalRead lê o estado do botão
                        // retorna um booleano
                        botao_pressionado = digitalRead(6); 
                        if (botao_pressionado == true) { // se for pressionado
                            digitalWrite(4, HIGH); // liga o led
                            delay(1000); // delay de 1 segundo
                        } else { // se não
                            digitalWrite(4, LOW); // desliga o led
                            delay(1000); // delay de 1 segundo
                        }
                        }

Funções:

pinMode

pinMode(pin, mode);
pin = Número do pino que deseja configurar
mode = se é entrada (INPUT) ou saída (OUTPUT)

Inputs: Sensores, botões

Outputs: Leds

digitalRead

A função digitalRead() no Arduino é usada para ler o estado lógico (HIGH or LOW) de um pino digital.

digitalRead(pin);

pin = número do pino digital.

digitalWrite

A função digitalWrite() no Arduino é usada para definir o estado lógico (alto ou baixo) de um pino digital específico.

digitalWrite(pin, value);

pin = número do pino digital.
value = valor que deseja mandar para o pino (HIGH or LOW)
